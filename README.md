## Overview

This project implements a Finite Impulse Response (FIR) digital filter on an FPGA using the PYNQ-Z2 board (Zynq-7020).
The filter is written in Verilog, integrated using Vivado Block Design, and controlled from Python in Jupyter Notebook.

The project demonstrates hardware–software co-design where:

Python generates input signals

FPGA performs FIR filtering

Python visualizes results in time and frequency domains
